Um serviço que monitora containers Docker e expõe dados via API REST.

## 🚀 Recursos

- 📊 Estatísticas de CPU, memória, rede e disco dos containers
- 🌍 API REST acessível via HTTP
- 🐳 Fácil deploy com Docker Compose
